=====
Input:  The system allows data producers to upload data into special storage area dedicated to them.
Process:  The system processes the data and transforms it into a meaningful database format; thus, making the data ready for consumption.
Output: The system allows data consumer to use industry standard web services to consume
The system should be hosted as a cloud base application that allows users with minimal IT or engineering skills to publish their data through the use of simple templates.
